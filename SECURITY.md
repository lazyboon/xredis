# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue first.

Please report it privately with:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Suggested fix (if available)

Until a dedicated security contact is configured, open a private channel with the maintainer and include "xredis security" in the subject.

## Supported Versions

Security fixes are provided on a best-effort basis for the latest release.
